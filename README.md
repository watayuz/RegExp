### "text".match("探す文字列")するとmatchした部分が返ってくる  

### "text".match(/[0-9]+/g)すると、そのtextが数字であるか調べられる

### 例
str = "12:34:56";  
if( str.match(/^[0-9][0-9]:[0-9][0-9]:[0-9][0-9]$/) ){ /マッチした時の処理/ }  
  
"text".match("regStr") // regStrを含んでいるかどうか  
var re = new RegExp("abc", "i");  
// iを指定で大文字小文字の区別をしない  
// gを指定で文字列の全体を検索できる
// mを指定で複数行に分けて検索できる
  
"abcdefg".match(re);
